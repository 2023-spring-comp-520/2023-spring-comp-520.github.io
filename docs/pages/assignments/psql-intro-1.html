PostgreSQL Intro 1

Intent: Download & Load the sample Database. First queries.
Submission: Take screenshots of the success of each step, and issue queries against the information schema in order to demonstrate SQL based results of having explored the DB objects.
  1. Visit PostgreSQL Tutorial Links and get setup, and if quickly successful - consider going on to Section 1 (see bottom).
  2. Connect to a PG DB Server or pgAdmin(4) (ie don't go on to "Connect to PG database from other Applications")
  3. Download Sample DB
  4. Load Sample DB
    • You will need to set the Bin Path when using pgAdmin in order to successfully Restore the DVDRental extract. Here's a hint: https://stackoverflow.com/questions/39408061/postgresql-binary-path-in-pgadmin-4-1-0-rc1
    • When loading the sample database, note that in the bottom right corner of the window modal which you use to find the extracted dvdrental tar file, the default is for Custom Files and it won't find the extracted file (no file extension), thus it won't see the file! Change the drop down to All Files, and your dvdrental extracted file should appear.
  5. Explore Server and DB Objects

Section 1 - Querying Data

  1. Select - shows you how to query data from a single table
  2. Order By - guides you on how to sort the result set returned from a query.
  3. Select Distinct - provides you a clause that removes duplicate rows in the result set.
